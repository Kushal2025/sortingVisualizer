Sorting Visualizer

Bubble Sort

1. For i from 0 to n-1:
2.    For j from 0 to n-i-1:
3.        If array[j] > array[j+1]:
4.            Swap(array[j], array[j+1])

Time Complexity:

Best: O(n)

Average: O(n²)

Worst: O(n²)

Selection Sort

1. For i from 0 to n-1:
2.    Set minIndex = i
3.    For j from i+1 to n:
4.        If array[j] < array[minIndex]:
5.            Set minIndex = j
6.    Swap(array[i], array[minIndex])

Time Complexity:

Best: O(n²)

Average: O(n²)

Worst: O(n²)

Insertion Sort

1. For i from 1 to n-1:
2.    key = array[i]
3.    j = i - 1
4.    While j >= 0 and array[j] > key:
5.        array[j + 1] = array[j]
6.        j--
7.    array[j + 1] = key

Time Complexity:

Best: O(n)

Average: O(n²)

Worst: O(n²)
